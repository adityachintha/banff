# Assets

No external image assets are required for this guide. The PDF generator uses a text-based EscapeWith.Ady brand mark and simple built-in PDF fonts so the file works offline.

If you add a real logo later, place it in this folder and update `scripts/generate-pdf.py`.

The `thumb-*.svg` files are lightweight local guide illustrations used by the website timeline. They avoid external image dependencies so the guide remains print/PDF friendly.
